需要jdk的rpm包，没有的话去官网下载

https://www.oracle.com/java/technologies/downloads/




#检测是否存在jdk

#查看jdk版本

java -version

#查看已经安装的jdk，如果是openjdk，则需要删掉

rpm -qa|grep jdk


#如果有openjdk，则需要删掉，删掉后安装需要的jdk


#强制卸载删除命令

#rpm -e --nodeps XXXXXXXX




#安装jdk

rpm -ivh jdk-XXXXX



#安装后查看是否安装成功

java -verison 和 java


注：rpm方式安装jdk不需要配置环境变量！！！
