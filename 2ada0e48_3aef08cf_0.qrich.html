statements


Python 的 for 语句不迭代算术递增数值，或是给予用户定义迭代步骤和暂停条件的能力（比如C语言），而是迭代列表或字符串等任意序列，元素的迭代顺序与在序列中出现的顺序一致。


遍历集合时修改集合的内容，会很容易生成错误的结果。因此不能直接进行循环，而是应遍历该集合的副本或创建新的集合


# Create a sample collection

users = {'Hans': 'active', 'Éléonore': 'inactive', '景太郎': 'active'}


# Strategy: Iterate over a copy

for user, status in users.copy().items():

if status == 'inactive':

del users[user]


# Strategy: Create a new collection

active_users = {}

for user, status in users.items():

if status == 'active':

active_users[user] = status

range()函数 生成数字序列

for x in range(a): #1,2,...,a-1

for x in range(a,b): #a,b区间包左不包右

for x in range(a,b,c) #c是步长


可以和 len() 函数结合遍历sequence下标

for i in range(len(seq)):


还可以使用 range()函数创建一个列表

list(range(5))


break语句用于跳出循环

continue语句用于跳出当前循环，继续下次循环


循环语句可以有 else 子句，它在穷尽列表(以for循环) 或条件变为 false (以while循环) 导致循环终止时被执行，但循环被 break 终止时不执行。
