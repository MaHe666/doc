@RequestMapping("/helloworld2")

public String helloworld2(){

//资源名可使用任意有业务语义的字符

if (SphO.entry("HelloWorld2")){

//务必保证finally会被执行

try{

/*

* 被保护的业务逻辑

*/

return "Hello,Sentinel! this is boolean response"+System.currentTimeMillis();

}finally {

SphO.exit();

}

} else {

//资源访问阻止，被限流或被降级

//进行相应的处理操作

return "系统繁忙，请稍后";

}

}





注：使用的HelloWorld2流控规则和HelloWorld的规则一样


SphO 提供 if-else 风格的API。用这种方式，当资源发生了限流之后会返回 false，这个时候可以根据返回值，进行限流之后的逻辑处理


注意！SphO.entry(xxxx) 需要和SphO.exit()方法成对出现，匹配调用，位置正确。

否则会导致调用链记录异常，抛出ErrorEntryFreeException
