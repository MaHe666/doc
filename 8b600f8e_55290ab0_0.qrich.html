Point类包含一个构造函数。

https://docs.oracle.com/javase/tutorial/java/javaOO/examples/Point.java

您可以识别构造函数，因为它的声明使用与类相同的名称，并且没有返回类型。

Point类中的构造函数接受两个整数参数，如代码所声明的（int a，int b）。

以下语句提供了23和94作为这些参数的值

Point originOne = new Point(23, 94);

而Rectangle类包含了四个构造方法

https://docs.oracle.com/javase/tutorial/java/javaOO/examples/Rectangle.java


每个构造函数都允许您使用基本类型和引用类型为矩形的原点、宽度和高度提供初始值。如果一个类有多个构造函数，它们必须有不同的签名。Java编译器根据参数的数量和类型区分构造函数。当Java编译器遇到以下代码时，它知道调用Rectangle类中的构造函数，该构造函数需要Point参数后跟两个整数参数：

Rectangle rectOne = new Rectangle(originOne, 100, 200);


这将调用Rectangle的一个构造函数，该构造函数将origin初始化为originOne。此外，构造函数将宽度设置为100，高度设置为200。现在有两个对同一Point对象的引用，一个对象可以有多个对它的引用，如下图所示：


以下代码行调用Rectangle构造函数，该构造函数需要两个整数参数，它们提供宽度和高度的初始值。如果检查构造函数中的代码，您将看到它创建了一个新的Point对象，其x和y值初始化为0：

Rectangle rectTwo = new Rectangle(50, 100);


以下语句中使用的Rectangle构造函数不接受任何参数，因此称为无参数构造函数（简称无参构造）：

Rectangle rect = new Rectangle();


所有类都至少有一个构造函数。如果类没有显式声明any，Java编译器会自动提供一个无参数构造函数，称为默认构造函数。此默认构造函数调用类父级的无参数构造函数，如果类没有其他父级，则调用Object构造函数。如果父级没有构造函数（Object没有构造函数），编译器将拒绝该程序。
