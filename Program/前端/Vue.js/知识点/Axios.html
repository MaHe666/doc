                      Axios

                      Mahe666

                      Axios是一个开源的可以用在浏览器端和 Node.js 的异步通信框架， 它的主要作用就是实现 ajax 异步通信

                      GitHub地址：https://github.com/axios/axiosopen in new window

                      中文官网：http://www.axios-js.com/open in new window

                      CDN

                      <script src="https://unpkg.com/axios/dist/axios.min.js"></script>

                      下载Axios

                      npm i axios -S
