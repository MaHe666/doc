项目里 application.yml 添加注册信息配置

seata:

registry:

type: nacos

nacos:

application: seata-server

server-addr: 127.0.0.1:8848

group : "SEATA_GROUP"

namespace: ""

username: "nacos"

password: "nacos"





seata-server端配置注册中心

在 registry.conf 中加入对应配置中心

registry {

type = "nacos"

nacos {

application = "seata-server"

serverAddr = "127.0.0.1:8848"

group = "SEATA_GROUP"

namespace = ""

cluster = "default"

username = ""

password = ""

}

}
