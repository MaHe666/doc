          SpringMVC

          Mahe666

          SpringMVC文档地址：https://docs.spring.io/spring-framework/docs/current/reference/html/web.html#spring-webopen in new window

          Spring的web框架围绕 DispatcherServlet 设计，DispatcherServlet 的作用是将请求分发到不同的处理器，是整个SpringMVC的控制中心

          DispatcherServlet是中心控制器，它的本质也是个Servlet

          用户发出的请求都会被 DispatcherServlet 接收并拦截
