package com.mahe666.lock8;


import java.util.concurrent.TimeUnit;


/**

* 1，标准情况下，两个线程先执行 method1 还是 method2？ method1 -> method2

* 2，method1方法延迟4秒，两个线程先执行 method1 还是 method2？ method1 -> method2

* 3，增加了一个普通方法method3后，是先执行method1还是method3？ method3 -> method1

* 4，创建两个资源类的实例化对象，分别执行两个同步方法。先执行method1还是method2? method2 -> method1

* 5，增加两个静态同步方法method4和method5。先执行method4还是method5? method4 -> method5

*/

public class Test5 {

public static void main(String[] args) {


Test5Resource test5Resource = new Test5Resource();


new Thread(()->{

test5Resource.method4();

},"A").start();


new Thread(()->{

test5Resource.method5();

},"B").start();


}

}


class Test5Resource{

public synchronized void method1(){

try {

TimeUnit.SECONDS.sleep(1);

} catch (InterruptedException e) {

e.printStackTrace();

}

System.out.println("method1");

}


public synchronized void method2(){

System.out.println("method2");

}


public void method3(){

System.out.println("method3");

}


/*

静态方法是在类加载的时候就有了，所以这里锁的并不是对象，而是Class模板。

而这里锁定的是当前这个类Test5Resource的模板，也就是这个类的反射对象。


Class<Test5Resource> test5ResourceClass = Test5Resource.class;

而且这个Class对象全局唯一，所以也就只有一个锁。这里锁的是Class反射对象


*/

public static synchronized void method4(){

try {

TimeUnit.SECONDS.sleep(1);

} catch (InterruptedException e) {

e.printStackTrace();

}

System.out.println("method4");

}


public static synchronized void method5(){

System.out.println("method5");

}

}
