多态的字典定义是指生物学中的一个原则，其中一个生物体或物种可以有许多不同的形式或阶段。这一原则也适用于面向对象编程和Java等语言。类的子类可以定义自己独特的行为，但也可以共享父类的一些相同功能。


多态性可以通过对超类的轻微修改来证明。例如，可以将超类的方法添加到显示实例中当前存储的所有数据的类中。

public class TestSuperClass {

public void sayHelloWorld(){

System.out.println("HelloWorld");

}

}


要演示Java语言中的多态特性，这里使用两个子类扩展了TestSuperClass类，使其有不同的实现

public class TestSubClass1 extends TestSuperClass{

private String name;


public TestSubClass1(String name) {

this.name = name;

}


@Override

public void sayHelloWorld(){

System.out.println("HelloWorld:" + name);

}

}


public class TestSubClass2 extends TestSuperClass{

private int number;


public TestSubClass2(int number) {

this.number = number;

}


@Override

public void sayHelloWorld(){

System.out.println("HelloWorld:" + number);

}

}


总的来说，有三个类别：TestSuperClass、TestSubClass1、TestSubClass2

其中的两个子类重写父类的方法，并按照自己的“方式”去打印信息


下面是测试程序

public class Main extends TestSuperClass{

public static void main(String[] args) {

TestSuperClass t1, t2, t3;

t1 = new TestSuperClass();

t2 = new TestSubClass1("mahe666");

t3 = new TestSubClass2(666);


t1.sayHelloWorld();

t2.sayHelloWorld();

t3.sayHelloWorld();

}

}


控制台打印如下信息：

HelloWorld

HelloWorld:mahe666

HelloWorld:666


Java虚拟机（JVM）为每个变量中引用的对象调用适当的方法。它不会调用由变量类型定义的方法。这种行为被称为虚拟方法调用，它展示了Java语言中重要多态特性的一个方面。
