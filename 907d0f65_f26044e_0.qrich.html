在项目中配置webpack


在项目根目录中创建名为 webpack.config.js 的webpack配置文件，并初始化如下的基本配置：

module.export = {

//开发的时候用development

//上线的时候用production

mode:"development" //mode代表构建模式（webpack的运行模式），可选值有development和production


}



"scripts":{
"dev":"webpack" //scripts下的节点脚本，可以通过npm run运行，例如npm run dev

}



运行npm run dev
