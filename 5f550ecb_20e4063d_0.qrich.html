乐观锁和悲观锁


乐观锁：

顾名思义，十分乐观，总是认为不会出现问题，无论干什么都先不上锁，如果出现了问题，就更新值测试



悲观锁：

顾名思义，十分悲观，总是认为会出现问题，无论干什么都会上锁，再去操作




乐观锁，官网地址：

https://mp.baomidou.com/guide/interceptor-optimistic-locker.html




乐观锁机制

当要更新一条记录的时候，希望这条记录没有被别人更新

乐观锁实现方式：

  • 取出记录时，获取当前version
  • 更新时，带上这个version
  • 执行更新时， set version = newVersion where version = oldVersion
  • 如果version不对，就更新失败




多线程的时候一定要用乐观锁！！！
