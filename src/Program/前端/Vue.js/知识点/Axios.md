`Axios`是一个开源的可以用在浏览器端和 `Node.js` 的异步通信框架， 它的主要作用就是实现 `ajax` 异步通信

GitHub地址：<https://github.com/axios/axios>

中文官网：<http://www.axios-js.com/>

**CDN**

```html
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
```

下载Axios

```bash
npm i axios -S
```
