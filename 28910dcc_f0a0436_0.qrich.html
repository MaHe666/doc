列表的数据项可以保存不同类型的元素

序列中的每个值都有对应的位置值，称之为索引，第一个索引是 0，第二个索引是 1，依此类推

列表是个可变序列，可以保存重复的数据，支持切片

创建一个列表，只要把逗号分隔的不同的数据项使用方括号[]括起来即可。

空列表list=[]


list = ['HelloWorld',666]



访问列表里面的值

print( list[0] )


更新列表元素

list[1] = 123


插入新列表元素

list.append("我爱中国")


删除列表元素

del list[0]



Python列表脚本操作符

列表对 + 和 * 的操作符与字符串相似。+ 号用于组合列表，* 号用于重复列表。


len(list) list的长度

list1 + list2 list拼接

list * n list重复n次

x in list 判断元素x是否在list内，如果在则true，否则false

for x in list: 将list的值循环赋给x，（相当于Java中的增强for循环）




list(seq) 将元组转换为列表
