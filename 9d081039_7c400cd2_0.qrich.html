1，你接收的参数是name，但是前端传过来一个username

解决方案，使用@RequestParam注解，建议基本类型参数都这么用


@RequestMapping("/test1")

public String test1(@RequestParam("username") String name, Model model){

model.addAttribute("msg",name);

return "test";

}




2，前端传了一个对象中的属性（例如用户注册表单等），

解决方案，后端没必要去接收每个参数，放一个对应的对象来接收就好。建议是vo对象，否则会浪费资源


@RequestMapping("/test2")

//http://localhost:8080/test2?username=mahe666&password=12345

public String test2(User user){

System.out.println(user);

return "test";

}
