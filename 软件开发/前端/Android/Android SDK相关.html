                  如果只需要下载 Android SDK，就在下载页面的底部找到 仅限命令行工具 (Command Line Tools only)，然后根据系统选择对应的文件下载

                  注意！下载后的文件解压缩后，只是单纯的 命令行工具，并不是 Android SDK

                  如果不习惯使用命令行，可以在国内官网首页上找到 SDK Tools 下载安装包

                  安装包安装后是可视化界面，会更方便一些

                  注意！使用安装包之前，需要安装好JDK

                  我为了偷懒，我使用的也是 installer版，笔记中记录的也是这个版本

                  具体安装方式，可参考：https://cloud.tencent.com/developer/article/2107040open in new window

                  下载地址：https://www.androiddevtools.cn/open in new window

                  配置环境变量

                  ANDROID_HOME: installer版安装后的 文件夹所在路径，例如：D:\myEnvironments\android_sdk

                  Path: 需要配置如下两个

                  • %ANDROID_HOME%\platform-tools
                  • %ANDROID_HOME%\tools

                  注意，配置这个 Path 的原因是需要直接访问 platform-tools 文件夹下的 adb.exe 文件

                  配置完环境变量后，打开cmd，输入 adb，如果出现一堆英文说明，则说明配置成功了

                  cmdline-tools 命令行工具

                  如果可能的话，还是使用这个比较好

                  命令行工具概览：https://developer.android.com/tools?hl=zh-cnopen in new window

                  安装 Android SDK 需要用到命令行工具下的 bin 目录里的 cmdline-tools\bin\sdkmanager 文件

                  SDK Manager

                  用来管理Android SDK，类似于nvm的作用

                  sdkmanager 说明；https://developer.android.com/tools/sdkmanager?hl=zh-cnopen in new window

                  关于镜像

                  国内如果没有梯子，可能无法访问Google，所以就需要镜像

                  在导航栏找到Tools -> Options -> HTTP Proxy Server 和 HTTP Proxy Port

                  HTTP Proxy Server 和 HTTP Proxy Port 的设置 可参考：https://cloud.tencent.com/developer/article/2100891open in new window

                  关于下载

                  需要下载的东西如下：

                  • Tools -> Android SDK Platform-tools、Android SDK Build-tools

                  • Android XXX(XXX代表了版本，可以指定版本，也可以下载最新版) -> 所有

                  • Extras -> 所有

                  AVD Manager

                  用来创建Android虚拟机，速度慢的一批

                  avdmanager 说明：https://developer.android.com/tools/avdmanager?hl=zh-cnopen in new window

                  使用

                  点击Create按钮，创建一个新的 Android Virtual Device

                  如图，每个选项的说明如下：

                  • AVD Name: 安卓虚拟机的名字
                  • Device: 设备分辨率等内容，不建议太大，电脑会卡顿
                  • Target：指定Android镜像版本，该虚拟机的操作系统也就是选定的版本
                  • CPU/ABI: 指定用来模拟的CPU，需要根据电脑的CPU进行选择
                  • Keyboard: 使用硬件键盘（就是电脑的键盘）
                  • Skin: 花里胡哨，不建议用
                  • Front Camera: 前置摄像头
                  • Back Camera: 后置摄像头
                  • Memory Options：安卓虚拟机的运存
                  • Internal Storage: 手机内部存储
                  • SD Card：手机外部存储（旧的那种储存卡）

                  启动之前时需要关闭 Hyper-v，在cmd中输入以下命令

                  bcdedit /set hypervisorlaunchtype off
