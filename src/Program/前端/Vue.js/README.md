`Vue` 只关心视图层（HTML+CSS+JS）,网络通信什么的都不需要他管

`Vue` 的开发基于NodeJS，实际开发采用vue-cli脚手架开发

`vue2` 中文官网：<https://v2.cn.vuejs.org/>

`vue3` 中文官网：<https://cn.vuejs.org/>

**安装Node.js淘宝镜像加速器**

```bash
npm install cnpm -g
```

**确认Node.js安装成功**

```bash
# 查看node版本
node -v
# 查看npm版本
npm -v
```
