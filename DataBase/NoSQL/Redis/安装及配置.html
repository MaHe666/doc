                    安装及配置

                    Mahe666

                    下载之后使用远程连接工具上传到Linux系统中

                    使用命令解压压缩包，然后把解压出来的文件夹放到/opt中

                    tar -zxvf XXXX

                    redis文件夹的目录下有个redis.conf文件，这个就是redis的配置文件了

                    在安装之前，我们需要做一些准备工作，配置一下环境。注意！需要联网

                    yum -y install gcc gcc-c++

                    然后在redis目录下使用 make 和 make install 命令

                    进入到redis的默认安装路径 /usr/local/bin 下，如果可以看到redis应用，就证明安装成功了

                    在redis的安装目录下新建一个用来装配置文件的文件夹，然后将解压出来的redis文件夹中的 redis.conf 文件复制到这里

                    因为，redis默认不是后台启动，所以我们需要修改一下配置。之后我们在运行redis时，用的就是这个配置文件

                    使用vim编辑redis.conf文件，将属性daemonize no改为 daemonize yes ，编辑好后保存退出

                    需要注意的是，redis的配置文件中指定了redis的端口为6379，这个端口我们在连接时会用到

                    回到redis的安装目录（默认是 /usr/local/bin ），输入命令运行redis，注意不要忘记指定redis配置文件的位置

                    redis-server myConfig/redis.conf

                    如果没有报错，则说明启动成功

                    输入命令启动redis客户端连接服务端测试

                    redis-cli -p 6379

                    连接后，光标前的显示会发生改变

                    测试是否连接成功，输入 ping 回车

                    下面是一些redis的基本命令：设置键值对，根据键获取值，查看所有存在的键

                    在这个时候，可以单开一个回话查看当前的redis进程

                    ps -ef|grep redis

                    想要退出redis及客户端，需要先关闭redis-server，再退出
