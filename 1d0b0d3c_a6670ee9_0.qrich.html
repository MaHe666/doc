数值

整数

tinyint 非常小的数据 1字节

smallint 较小的数据 2字节

mediumint 中等大小的数据 3字节

int 标准的整数 4字节 int 常用

bigint 较大的数据 8字节 long

浮点数

float 单精度 4字节 float

double 双精度 8字节 double

decimal 字符串形式的浮点数 金融计算的时候，一般使用decimal



字符串


char 固定大小的字符串 0-255

varchar 可变字符串 0-65535 string 常用

tinytext 微型文本 2^8-1个字（255）

text 文本串 2^16-1个字（65535） 保存大文本


时间日期


java.util.Date

date yyyy-MM-dd 日期格式

time HH:mm:ss 时间格式

datetime yyyy-MM-dd HH:mm:ss 日期时间格式 最常用的

timestamp 1970.1.1 到现在的毫秒数 时间戳 也较为常用


year 年份表示


null


没有值，未知 注意，不要使用NULL进行运算，结果为NULL
