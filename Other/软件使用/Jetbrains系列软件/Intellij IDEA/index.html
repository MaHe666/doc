                                              # 添加本地Jar包

                                              idea添加本地jar包方法：

                                              1. 在与src同级的位置新建lib文件夹

                                              2. 放入jar包

                                              3. 然后右键lib文件夹，选择 Add as Library

                                              当需要添加其他包时

                                              需要点击这个去掉旧的Library，然后重新 Add as Library
