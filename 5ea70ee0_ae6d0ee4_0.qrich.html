若已知具体的类，通过类的class的属性获取，该方法最为安全可靠，程序性能最高

Class class = Person.class;


已知某个类的实例，调用该实例的getClass()方法获取Class对象

Class class = person.getClass();


已知一个类的全类名，且该类在类的类路径下，可通过Class类的静态方法forName()获取，可能抛出ClassNotFoundException

Class class = Class.forName("demo1.Person");

例如：JDBC的 Class.forName("com.mysql.jdbc.Driver");


内置基本数据类型可以直接用类名.TYPE

Class class = Integer.TYPE


还可以利用ClassLoader等



获取父类类型

Class class = person.getSuperclass();
