建议使用压缩包，不要使用exe安装包，因为他会走到注册表，之后换版本会很麻烦


1，解压

2，将包放一个方便的位置以进行后续的安装

3，配置环境变量

将mysql的bin目录路径放到系统环境变量的path里面

4，新建mysql配置文件 my.ini

5，启动管理员模式下的cmd，进到bin目录，

6，安装mysql服务

mysqld -install

7，初始化数据库文件

mysqld --initialize-insecure --user=mysql

8，启动mysql服务，输入

net start mysql

9，通过命令行进入mysql，输入

mysql -u root -p #-p后面不要带空格

10，修改密码

update mysql.user set authentication_string=password('密码') where user='root' and Host='localhost';

11，刷新权限

flush privileges;

12，删除或注释 my.ini 最后一行的 skip-grant-tables #为注释

13，重启mysql服务

net stop mysql

net start mysql
