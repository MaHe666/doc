### 安全equals

```java
private boolean safeEqual(String a, String b){
    if (a.length() != b.length()){
        return false;
    }
    int equal = 0;
    for (int i = 0; i < a.length(); i++) {
        equal |= a.charAt(i) ^ b.charAt(i);
    }
    return equal == 0;
}
```
