堆中的每一个对象都有一个引用计数，被引用一次，计数+1，被引用变量值为null，则计数-1，直到计数为0，则表示变为无用对象。


优点是算法简单，缺点是“循环引用的无用对象”无法被识别


循环引用示例

public class Student {

String name;

Student friend;


public static void main(String[] args) {

Student s1 = new Student();

Student s2 = new Student();


s1.friend = s2;

s2.friend = s1;


s1 = null;

s2 = null;

}

}


s1和s2这两个对象互相引用，导致他们的计数不为零，实际已经无用，但无法被识别
