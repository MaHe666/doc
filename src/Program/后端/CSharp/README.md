## .NET

下载适用于VS Code的 .Net SDK：<https://dotnet.microsoft.com/zh-cn/download/dotnet/sdk-for-vs-code>

> `.NET` 分为两种：`.NET Framework` 和 `.NET Core`
>
> `.NET Framework` 不能跨平台，只能在Windows系统上部署
>
> `.NET Core` 可以跨平台，是微软后来新出的框架
