例如，Point类是这样编写的

public class Point {

public int x = 0;

public int y = 0;

//constructor

public Point(int a, int b) {

x = a;

y = b;

}

}


但它可能是这样写的

public class Point {

public int x = 0;

public int y = 0;

//constructor

public Point(int x, int y) {

this.x = x;

this.y = y;

}

}


构造函数的每个参数都会隐藏对象的一个字段 —— 构造函数内的x是构造函数第一个参数的本地副本。要引用Point字段x，构造函数必须使用this.x。
