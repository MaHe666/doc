1，去github上下载

master分支上的dubbo-admin

https://github.com/apache/dubbo-admin.git


2，使用idea打开dubbo-admin文件夹（压缩包需要解压）

下载maven配置的jar包

编辑dubbo-admin-server模块下的application.properties配置文件


# 这里对注册中心相关地址进行相关配置，将端口号修改为Zookeeper的2181

admin.registry.address=zookeeper://127.0.0.1:2181

admin.config-center=zookeeper://127.0.0.1:2181

admin.metadata-report.address=zookeeper://127.0.0.1:2181

# 在配置文件末尾修改一下启动端口号，防止冲突

server.port=7001


3，打开idea中的maven面板，点击dubbo-admin-server模块，然后点击Lifecycle里面的package进行打包


4，然后回到项目目录，点击dubbo-admin-ui，右键选择Open in Terminal

输入npm i 让vue下载依赖

点开vue.config.js，更改proxy节点下target链接的端口，改为7001，和dubbo-admin-server暴露的端口一样

下载完依赖后，输入npm run dev运行


5，启动zookeeper和后端服务，点击控制台输出的端口进到页面，默认的账号和密码都是root

//可以在idea中运行后端服务，也可以在cmd中输入 java -jar 带后缀的文件名 运行服务
