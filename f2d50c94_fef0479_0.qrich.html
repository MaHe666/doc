捕获异常


try:

正常执行代码


一个except语句可以处理多个异常，这些异常通常被放在()内成为一个元组

except (异常1,异常2,...,异常n):

发生异常时执行的代码


也可以拆开写

except 异常1:


except 异常2:


最后一个异常可以忽略异常名，作为通配符使用

except:


如果没有发生任何异常，则触发else语句

使用 else 子句比把所有的语句都放在 try 子句里面要好，这样可以避免一些意想不到，而 except 又无法捕获的异常。

异常处理并不仅仅处理那些直接发生在 try 子句中的异常，而且还能处理子句中调用的函数（甚至间接调用的函数）里抛出的异常

else:

没有异常时执行的代码

finally:

不管有没异常都会执行







抛出异常

使用raise触发异常，raise 唯一的一个参数指定了要被抛出的异常。它必须是一个异常的实例或者是异常的类（也就是Exception的子类）

raise 异常类型('异常内容')


如果你只想知道这是否抛出了一个异常，并不想去处理它，那么一个简单的 raise 语句就可以再次把它抛出

>>> try:

raise NameError('HiThere')

except NameError:

print('An exception flew by!')

raise
