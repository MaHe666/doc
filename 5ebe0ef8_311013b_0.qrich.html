JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，目前使用特别广泛


JS示例


<script type="text/javascript">


//编写一个JavaScript对象

const user = {

name: "mahe666",

age: 3,

gender: "男"

};


console.log(user)


//将js对象转换为JSON字符串

var j = JSON.stringify(user)

console.log(j)


//将JSON字符串转换为js对象

var js = JSON.parse(j);

console.log(js)

</script>



花括号表示对象，方括号表示数组




Controller返回JSON数据
