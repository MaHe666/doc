## 设置git不使用SSL安全连接

```bash
git config --global http.sslVerify "false"

git config --global https.sslVerify "false"
```

## 设置允许上传的最大大小

```bash
git config --local http.postBuffer 1073741824

git config --local https.postBuffer 1073741824
```

## 查看commit详情

查看所有的commit记录
```bash
git log
```

查看提交的详情
```bash
git show
```

查看某次commit中对某个文件的修改
```bash
git show commit的id 文件名
```
