创建数组的一种方式是使用new关键字。

anArray = new int[10];

以上语句的意思是：为10个整数元素分配足够内存的数组，并将该数组分配给anArray变量


如果缺少这个过程，则编译器就会报错，提示该变量可能没有被初始化


还有更快捷的方式去创建和初始化数组:

int[] anArray = {100,200,300,400,500,600,700,800,900}

这里，数组的长度由大括号之间提供的值的数量决定，并用逗号分隔。


还可以通过使用两组或多组括号（如String[][]名称）来声明数组（也称为多维数组或二维数组）。因此，每个元素必须由相应数量的索引值访问。


class MultiDimArrayDemo {

public static void main(String[] args) {

String[][] names = {

{"Mr. ", "Mrs. ", "Ms. "},

{"Smith", "Jones"}

};

// Mr. Smith

System.out.println(names[0][0] + names[1][0]);

// Ms. Jones

System.out.println(names[0][2] + names[1][1]);

}

}


在Java编程语言中，多维数组是其组件本身就是数组的数组。
