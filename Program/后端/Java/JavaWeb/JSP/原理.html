                              • 流程

                              服务器内部工作：Tomcat中有一个work目录。IDEA中的代码使用Tomcat的会在IDEA的Tomcat工作空间生成一个work目录

                              我的Idea Tomcat的工作空间：C:\Users\ASUS\AppData\Local\JetBrains\IntelliJIdea2020.2\tomcat\Unnamed_javaweb-test\work\Catalina\localhost\cs\org\apache\jsp

                              打开其中的文件，发现页面转变成了java程序

                              浏览器向服务器发送请求，不管访问什么资源，其实都是在访问Servlet

                              JSP最终也会被转换成一个Java类，这个java类继承了HttpJspBase类，而HttpJspBase类又继承了HttpServlet类

                              JSP本质上就是一个Servlet
