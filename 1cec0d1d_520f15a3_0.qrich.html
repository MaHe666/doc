references 引用



方式一：在创建表的时候，添加外键约束


KEY `外键名` (绑定的字段),

CONSTRAINT `外键名` FOREIGN KEY (`字段名`) REFERNENCES `需要关联的表名` (`字段名`)



方式二：创建表成功后，添加外键约束

ALTER TABLE `表名` ADD

CONSTRAINT `外键名` FOREIGN KEY (`字段名`) REFERNENCES `需要关联的表名` (`字段名`)




以上操作的都是物理外键，数据库级别的外键，不建议使用，了解即可! (避免数据库过多造成困扰)

之后如果需要使用外键，则使用程序级别的外键



阿里的JAVA规范中有

【强制】 不得使用外键与级联，一切外键概念必须在应用层解决
