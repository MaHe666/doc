数据定义语言DDL用来创建数据库中的各种对象-----表、视图、索引、同义词、聚簇等如：CREATE TABLE / VIEW / INDEX / SYN / CLUSTER| 表 视图 索引 同义词 簇。DDL操作是隐性提交的！不能rollback


所有的语句都使用分号结尾


连接数据库

mysql -uroot -p #除外，结尾不能带空格


修改密码

update mysql.user set authentication_string=password('密码') where user='root' Host='localhost';


刷新权限

flush privileges;


查看所有数据库

show databases;


切换数据库

use 数据库名;


查看所有的表

show tables;


查看表的信息

describe 表名；


创建数据库

create database 数据库名；


退出连接

exit;
