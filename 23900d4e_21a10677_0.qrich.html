代码块(有时叫做复合语句)，是用花括号扩起的任意数量的简单Java语句。块确定了局部变量的作用域。块中的程序代码，作为一个整体，是要被一起执行的。块可以被嵌套在另一个块中，但是不能在两个嵌套的块内声明同名的变量。代码块可以使用外部的变量，而外部不能使用语句块中定义的变量，因为代码块中定义的变量作用域只限于代码块。



public class Test {

public static void main(String[] args) {

int a;

int b;

{

int c;

int a; // 编译错误，不能重复定义变量a

} // 变量c的作用域到此为止

}

}



这里的代码块没有名字，所以只能使用一次，并不常用
